# 🔐 Password Generator

A simple password generator built with React as part of my React learning journey.

## 📚 About This Project

This project was created to practice React concepts while building a useful application. It generates secure random passwords based on user-selected options.

## 🚀 Live Demo

https://ashmk04.github.io/Password-Generator/

## ✨ Features

- Generate random passwords instantly
- Adjustable password length
- Include numbers
- Include special characters
- Copy password to clipboard
- Responsive user interface

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript
- Tailwind CSS

## 📖 What I Learned

Through this project, I practiced:

- React Hooks (`useState`, `useCallback`, `useEffect`, `useRef`)
- State Management
- Event Handling
- Clipboard API
- Component-based UI development
- Deploying React applications with GitHub Pages

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Ashmk04/Password-Generator.git
```

Navigate to the project folder:

```bash
cd Password-Generator
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

## 🏗️ Build for Production

```bash
npm run build
```
